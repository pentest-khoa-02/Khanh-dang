## Promise 
 ### Promise là gì  ? 
  >  Promise sinh ra để xử lý các câu lệnh bất đồng bộ 

  ### Đầu tiên thì mình nói đến đồng bộ or bất đồng bộ trong js 

  ### Đồng bộ 
  ```
  console.log(1)
  console.log(2)
  console.log(3)

  output :  1 2 3  
  // nó sẽ chạy lần lượt từ trên xuống dưới 
  ```

  ### Bất đồng bộ 
  ```
  setTimeout(()=>console.log(1))
  console.log(2)
  console.log(3)
  
  output : 2 3 1
  ```
